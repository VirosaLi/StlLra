# STL-LRA Analysis

## Environment Setup

* Install miniconda from https://docs.conda.io/en/latest/miniconda.html
* Create conda environment

```shell script
    conda update conda
    conda create --name data python=3.8 pandas requests xlrd beautifulsoup4
```

* Install LibreOffice
    * Linux
    ```shell script
      sudo snap install libreoffice
    ```
